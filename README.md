# Quantum Sailing Routing 
A little project combining Ising model and Quantum Mechanics using Qiskit through a Quantum Approximate Optimization Algorithm.
The goal of this algorithm is to compute the best way for a sailing boat to go from a point to another taking into account the wind, the 
swell, the current, the depth
As there is already existing algorithm that does the same thing, the goal of this project is just to gain experience in two hobbies : quantum computing and sailing.
